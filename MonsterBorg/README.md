# Monsterbord steering
## Get started

1. Create and activate virtual environment
2. install dependencies using `pip3 install ..`
3. set environmental variables in your terminal
```sh
export ROBOT_HOST="your_api_key_here" # optional, otherwise 0.0.0.0
export ROBOT_API_KEY="your_api_key_here"
```
4. start listener by running `python3 listener.py`
5. start april tag location detector by running `python3 pos_detector.py`
